================================================================================
                    TROUBLESHOOTING
================================================================================

ISSUE: cargo command not found
SOLUTION: Install Rust from https://rustup.rs/
          Then restart terminal and run: source $HOME/.cargo/env

ISSUE: Compilation errors in Rust code
SOLUTION: Run: cargo clean
          Then: cargo build --release
          Check error messages for missing dependencies

ISSUE: Module not found errors
SOLUTION: Ensure all .rs files are in src/ directory:
          - cobol_parser_enhanced.rs
          - fortran_generator_enhanced.rs
          - sql_translator.rs
          - copybook_handler.rs
          - sort_merge_handler.rs
          - decimal_arithmetic.rs
          - enterprise_error_handler.rs

ISSUE: GFortran compilation errors
SOLUTION: Check Fortran code syntax
          Verify GFortran version (9.0+)
          Review generated Fortran code

ISSUE: Runtime errors in Fortran program
SOLUTION: Check input data format
          Verify file paths
          Review COBOL-to-Fortran translation
          Enable Fortran runtime checks: gfortran -fcheck=all
