================================================================================
                        FORTRAN COMPILATION
================================================================================

COMPILE GENERATED FORTRAN:
  gfortran output.f90 -o program

COMPILE WITH DEBUGGING:
  gfortran -g output.f90 -o program

COMPILE WITH OPTIMIZATION:
  gfortran -O3 output.f90 -o program

COMPILE WITH RUNTIME CHECKS:
  gfortran -fcheck=all output.f90 -o program

RUN FORTRAN PROGRAM:
  ./program

================================================================================
                        COMPILATION PIPELINE
================================================================================

Phase 1: Parse COBOL code
         → Enhanced Data Division support
         → Symbol table generation
         → Type inference

Phase 2: Process COPY statements
         → Search multiple paths
         → Apply REPLACING clauses

Phase 3: Translate EXEC SQL blocks
         → Parse SQL statements
         → Generate Fortran SQL calls

Phase 4: Process SORT/MERGE operations
         → Detect SORT statements
         → Generate sorting logic

Phase 5: Generate Fortran code
         → DECIMAL_ARITHMETIC module
         → ERROR_HANDLER module
         → Main program code

Phase 6: Write output file
         → Format code
         → Generate report

                        SUPPORTED COBOL FEATURES
================================================================================

DATA DIVISION:
  ✓ PIC 9(n)              - Numeric
  ✓ PIC 9(n)V9(m)         - Decimal
  ✓ PIC X(n)              - Alphanumeric
  ✓ PIC A(n)              - Alphabetic
  ✓ PIC S9(n)             - Signed numeric
  ✓ PIC Z(n).99           - Edited numeric
  ✓ COMP, COMP-3          - Computational
  ✓ OCCURS n TIMES        - Arrays
  ✓ REDEFINES             - Data redefinition
  ✓ VALUE clause          - Initial values

PROCEDURE DIVISION:
  ✓ MOVE                  - Data movement
  ✓ COMPUTE               - Arithmetic expressions
  ✓ ADD, SUBTRACT         - Arithmetic operations
  ✓ MULTIPLY, DIVIDE      - More arithmetic
  ✓ IF-THEN-ELSE          - Conditional logic

  ✓ PERFORM               - Subroutine calls
  ✓ DISPLAY               - Output
  ✓ ACCEPT                - Input

FILE OPERATIONS:
  ✓ OPEN                  - Open files
  ✓ CLOSE                 - Close files

BATCH OPERATIONS:

ERROR HANDLING:

================================================================================
                        TROUBLESHOOTING
================================================================================

PROBLEM: "cargo: command not found"
SOLUTION: Install Rust from https://rustup.rs/

PROBLEM: "error: could not compile"
SOLUTION: Run "cargo clean" then "cargo build --release"

PROBLEM: "module not found"
SOLUTION: Ensure all .rs files are in src/ directory

PROBLEM: "GFortran compilation failed"
SOLUTION: Check Fortran syntax, verify GFortran 9.0+

PROBLEM: "Precision errors in output"
SOLUTION: Enable exact precision mode, use DECIMAL_ARITHMETIC module

PROBLEM: "Parse error in COBOL"
SOLUTION: Check COBOL syntax, review TRANSLATION-EXAMPLES.txt

PROBLEM: "SORT not working"
SOLUTION: Check SORT syntax, ensure sort keys are defined

PROBLEM: "SQL translation failed"
SOLUTION: Verify EXEC SQL syntax, check database connectivity

                        PERFORMANCE TIPS
================================================================================

1. Use release build for production:
   cargo build --release (5-10x faster than debug)

2. Compile Fortran with optimization:
   gfortran -O3 output.f90 -o program

3. For large COBOL programs:
   - Split into smaller modules
   - Use COPY statements
   - Compile in batches

4. For batch processing:
   - Use SORT/MERGE efficiently
   - Process in chunks
   - Enable parallel processing

5. For banking applications:
   - Always use precision arithmetic
   - Enable audit trail
   - Log all errors
